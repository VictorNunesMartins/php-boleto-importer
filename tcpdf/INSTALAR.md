# Como instalar o TCPDF

Esta pasta deve conter os arquivos do TCPDF.

## Download

1. Acesse: https://github.com/tecnickcom/TCPDF/releases
2. Baixe o ZIP da versão mais recente
3. Extraia **todo o conteúdo** dentro desta pasta `tcpdf/`

## Estrutura esperada após extração

```
tcpdf/
├── tcpdf.php          ← arquivo principal (incluído pelo gerador)
├── config/
├── fonts/
├── include/
└── ...
```

## Verificação

Se o arquivo `tcpdf/tcpdf.php` existir, o gerador vai funcionar.
